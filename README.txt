<パンジャンドラム> {
英国紳士の嗜みこと、軽くてかっこいいパンジャンドラムです。
配列複製やブーリアンといったBlenderの機能を行使しながらも
ポリゴン数や容量を軽く抑えることができ、
アレンジで各々の個性や好み、デザインのセンスもよく現れ、
なおかつ、スキルの向上と共に後からでも手を加えやすいという点を考えて
最初は機能に慣れるために、そして今は自己アピールのために作っています。
} 5.0 {
Github に載せる最初のパンジャンです。
とにかくわざとらしいほどに機能を使い、完成させることを考え、シンプルに仕上げました。
支柱12本で横から見ると時計のような形になっていて面白がって大きくした結果
ホビン部分がやや小さくなってしまい、横に転倒しやすそうなサイズになっているので
今はどうにかホビンの部分だけを縦横大きくできないか試作中です。
}
